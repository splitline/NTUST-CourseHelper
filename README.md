# NTUST-CourseHelper

一個協助台科人選課的小工具。 

## 功能/簡介

它能為台科課程查詢結果加入以下功能：
- 搜尋/過濾
- 針對欄位排序
- 一鍵Google「課名+教師」
- 判斷該門課是否還有名額
